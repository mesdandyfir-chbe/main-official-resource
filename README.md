# main-official-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源的形式存放，可通过仓库的 Pages 功能或直接访问原始文件进行浏览。

## 项目简介

这是一个以 HTML 文件为主要内容的存档仓库。页面内容覆盖多个独立主题，各自对应一个或多个 HTML 文件。仓库本身不绑定特定网站或域名，也不提供任何形式的导航或索引页面（除本 README 外）。

## 目录说明

```
main-official-resource/
├── README.md            # 本文件
├── index.html           # 仓库根目录的入口页面（如有）
├── page-a/              # 示例：某类页面的文件夹
│   ├── index.html
│   └── asset/
├── page-b/              # 示例：另一类页面的文件夹
│   ├── index.html
│   └── style.css
└── ...                  # 其他独立 HTML 文件或文件夹
```

- 每个文件夹通常代表一个独立的 HTML 页面或一组相关页面。
- 文件夹内可能包含 CSS、JavaScript、图片等附属资源。
- 部分页面可能直接位于根目录。

## 页面归档说明

- 所有 HTML 文件均为静态页面，不依赖后端服务。
- 每个页面独立运行，彼此之间没有依赖关系。
- 页面可能引用外部资源（如字体、图标库），这些资源来自其各自的公开 CDN。
- 不保证所有页面在所有浏览器或设备上完全兼容，但尽量遵循基本 Web 标准。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/main-official-resource.git
   ```
2. 直接在浏览器中打开目标 HTML 文件。
3. 若仓库已启用 GitHub Pages，可通过对应 URL 访问根目录或子目录下的页面。
